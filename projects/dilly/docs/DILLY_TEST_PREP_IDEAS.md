# Dilly Test Prep — ideas & wedge

Scratchpad for product narrative, hooks, and validation—**not** the main Meridian ideas list (`IDEAS.md`).

---

## Investor-style wedge (one paragraph)

Test prep is a high-stakes, high-willingness-to-pay category stuck in “infinite content + guilt.” Dilly meets students where they already live—for internships, deadlines, and voice coaching—and reframes prep as **short behavioral loops** (weak spot → explanation → micro-drill) that survive a real campus schedule, not a second full-time job. The moat is **distribution + habit inside the career OS** and **trust** (accuracy, no fake score precision), not another generic question bank; expansion is subscription during sprint windows, then optional B2B2C (orgs, cohorts) if engagement metrics hold.

---

## Product / GTM hooks

1. **Time-budget-first planner** — User sets minutes per day + exam date; product generates only what fits (no endless chapter map). Reduces abandonment from overwhelm.

2. **Panic / reset mode** — Five-minute flow: easiest wins + breathing copy + optional Voice; exits to a “real” session tomorrow. Converts anxiety into retention instead of app quit.

3. **Teach-back Voice loop** — User explains a problem or concept; Dilly probes gaps (Feynman-style). Complements multiple-choice without needing a huge item bank on day one.

4. **Wrong-answer archetypes** — Tag distractors and user errors (speed, careless, concept gap, pattern trap); analytics and copy focus on **why humans miss**, not only “B is correct.”

5. **Exam-week simulation** — Timer, breaks, phone-down / focus copy; boring but high perceived ROI; optional shareable “mock day” moment for social proof.

6. **Same-test-date cohorts (lightweight)** — Anonymous opt-in: others testing the same date + streak or weekly challenge; no full social graph.

7. **Weekly new-material cap** — Anti-hoarding: limit fresh topics per week; emphasize **retrieval** (cards, Voice recall) over passive reading.

8. **Pipeline link** — Surface test date next to application tracker / goals (“score goal unlocks this application wave”). One narrative: exam → school/job step.

9. **Honest score bands** — Ranges + “what this week of X usually moves” instead of false precision; aligns with truth-first positioning.

10. **Content strategy gate** — Before scale: choose official/licensed vs generated items; legal + accuracy review path so trust isn’t blown by one bad explanation.
