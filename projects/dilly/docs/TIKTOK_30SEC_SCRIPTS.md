**Rule:** Every second earns the next. No filler.  
**Text on screen:** Assume muted. Key lines should be readable.

## Production Notes

| Element | Recommendation |
|--------|----------------|
| **Hook** | First 1–2 sec = make or break. Text on screen must match the hook. |
| **Pacing** | Fast. 2–3 sec per beat. No dead air. |
| **Sound** | Use a trending sound for discoverability, or go silent + captions (many watch muted). |
| **Captions** | Burn them in. Don't rely on auto-captions. |
| **CTA** | "Link in bio" + point to bio. Repeat at end. |
| **Thumbnail** | Your face + bold text. "Your resume is worthless" or "300 million" or "47 applications" |
| **Hashtags** | #collegelife #jobsearch #resume #graduation #careeradvice #collegestudent #jobhunt #meridian |

## A/B Test Angles

- **Fear:** Scripts A, B, C (job crisis, resume dead)
- **Relatability:** Script D (I got ghosted, I built this)
- **Practical:** Script E (stop doing X, do Y instead)

Run 2–3 variants. Double down on what gets saves and shares — that's the conversion signal on TikTok.
