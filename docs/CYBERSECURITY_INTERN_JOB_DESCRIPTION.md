# Job description

## Position title

**Dilly — Trust & Security Intern**

## Role (one paragraph)

At Dilly we work with highly sensitive student data: resumes, AI conversations, rejection letters, and more, so trust has to be real. I’m looking for a Trust & Security Intern to help map where data lives, run guided security checks, write things up clearly, and pair on small fixes while we encrypt sensitive data and lay the groundwork for protecting every user as we grow. You don’t need to be a cybersecurity expert; this is a strong fit for a student who’s curious, careful with confidential info, and eager to learn. **Part-time (~15–20 hours/week), ~12 weeks, beginning Monday, May 12, 2026.** Because Dilly’s main work is in **New York**, the internship **may be remote**. The internship is **unpaid** (academic credit may be possible depending on your program; happy to discuss). If you’re on an **F-1 visa**, we can work with your school’s international office on **CPT** (or whatever your **DSO** requires) before you start. If you’re interested, send your resume and a short note on why this fits you and any relevant classes or projects. **We look forward to hearing from you.**
